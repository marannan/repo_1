Project 1 - ID3 Decision Tree
Build No: v1.0
Release Date: 10-05-2015
Author: Ashok Marannan <marannan@wisc.edu>

Introduction: 
	This program will generate ID3 decision tree based on training set and classsify testing set.

Installation:
	This package contains javabridge and weka-python-wrapper python packages in "Dependencies" directory.  
	Install them as necessary.

Usage: 
	python dt-learn.py <train-set-file> <test-set-file> m
	
Dependencies:
	Python 2.7
	Numpy
	javabridge https://pypi.python.org/pypi/javabridge (python-weka-wrapper depends on javabridge)
	weka-python-wrapper https://pypi.python.org/pypi/python-weka-wrapper (.arff file parser)
